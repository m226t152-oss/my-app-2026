# 英語学習アプリ (English Learning Web App)

日本人向けのシンプルで効果的な英語学習アプリケーションです。HTML, CSS, JavaScriptのみで構成されており、GitHub Pagesで簡単に公開できます。

## 主な機能
- **単語登録**: 新しい英単語と日本語訳のペアを保存。
- **単語一覧**: 登録した単語の確認と削除。
- **クイズ**: 登録した単語を使ったフラッシュカード形式の学習。
- **学習ログ**: 単語の追加やクイズの実施履歴を自動記録。
- **永続化**: データはブラウザの `localStorage` に保存されるため、ページを閉じても消えません。

## GitHub Pages での公開方法
1. GitHub で新しいリポジトリを作成します。
2. このプロジェクトのファイル (`index.html`, `css/`, `js/`) をそのリポジトリにプッシュします。
3. リポジトリの **Settings** > **Pages** に移動します。
4. **Build and deployment** > **Branch** で `main` ブランチを選択して **Save** をクリックします。
5. 数分後、発行されたURLでアプリが公開されます。

## 開発・カスタマイズ
- `index.html`: UI構造
- `css/style.css`: デザイン・スタイル
- `js/app.js`: ロジック・データ管理
